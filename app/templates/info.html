About L2C

L2C is a system for managing customer information, providing CRUD operations for customer data and MCP service interfaces.

API Endpoints

  • Customer Management

    • GET /api/customers - Get all customers
    • GET /api/customers/{id} - Get specific customer
    • POST /api/customers - Create new customer
    • PUT /api/customers/{id} - Update customer
    • DELETE /api/customers/{id} - Delete customer

  • MCP Service

    • POST /mcp - MCP query interface
